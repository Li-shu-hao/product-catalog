================================================
  精品商品目录 - 使用说明
================================================

📁 文件说明
-----------
index.html    - 主页面（双击用浏览器打开即可预览）
products.js   - 商品数据文件（用记事本编辑，添加/修改商品）
images/       - 图片文件夹（把你的商品图片放这里）
README.txt    - 本说明文件


📝 如何添加商品
--------------
1. 用记事本（或其他文本编辑器）打开 products.js
2. 照着现有格式复制一段，改成你的商品信息
3. 保存文件，刷新浏览器即可看到效果

每个商品格式如下：
{
  id: 数字编号（不要重复）,
  name: "商品名称",
  category: "分类名称",
  price: "价格（可选，如 ¥299）",
  code: "货号（可选）",
  image: "images/你的图片文件名.jpg",
  description: "简短描述（可选，用于搜索）",
  tags: ["标签1", "标签2"]  // 可选: 新品、热卖、特价、推荐
},

⚠ 注意:
- 分类名称要统一（比如都是"连衣裙"，不要一会写"连衣裙"一会写"裙子"）
- 图片路径相对 index.html，默认放在 images/ 文件夹里
- 标签目前支持: "新品" "热卖" "特价" "推荐"


🖼 图片准备建议
--------------
- 建议使用 3:4 比例竖版图（如 900x1200 像素），展示效果最好
- 也可以用 1:1 方图或 4:3 横图
- 图片质量建议 1500px 宽以上，压缩到 300KB 以内
- 图片命名建议用英文或拼音，避免中文文件名
- 支持的格式：JPG、PNG、WebP


🚀 如何部署上线（生成链接发给客人）
--------------
方式一：GitHub Pages（免费，推荐）
  1. 注册 GitHub 账号: https://github.com
  2. 新建仓库（New Repository），命名为 product-catalog
  3. 把整个 product-catalog 文件夹上传到仓库
  4. Settings → Pages → Source 选 main 分支 → Save
  5. 几分钟后会生成链接: https://你的用户名.github.io/product-catalog/
  6. 把这个链接发给客人即可

方式二：Netlify（免费，更简单）
  1. 注册 Netlify 账号: https://netlify.com
  2. 把整个 product-catalog 文件夹拖到 Netlify 页面
  3. 自动生成链接，可以自定义域名

方式三：云存储
  1. 把文件夹上传到阿里云OSS / 腾讯云COS
  2. 开启静态网站托管功能
  3. 生成访问链接


🔧 自定义修改
--------------
- 店铺名称：编辑 index.html，搜索"精品目录"替换成你的店名
- 主题色：编辑 index.html，找到 :root 里的 --color-accent，改成你的品牌色
- 联系方式：编辑 index.html，找到 footer 区域修改
- 微信分享封面：把你的封面图命名为 og-cover.jpg 放到 images/ 文件夹


💡 小技巧
--------
- 给客人发链接时可以带上分类参数：
  例如: https://你的域名/?cat=连衣裙
  客人打开直接看到连衣裙分类
- 搜索参数也可以带上：
  例如: https://你的域名/?q=碎花


❓ 常见问题
----------
Q: 双击 index.html 图片不显示？
A: 检查 products.js 里 image 路径是否正确，图片是否在 images/ 文件夹里

Q: 为什么搜索不到我的商品？
A: 搜索范围是商品名称、描述和标签，检查这些字段是否有对应关键词

Q: 怎么修改商品顺序？
A: 在 products.js 里调整商品的位置即可，排在前面显示在前面

Q: 手机上看效果不好？
A: 页面已适配手机，如果图片加载慢请压缩图片大小
